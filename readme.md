# DevCamper API

> Backed api for the DevCamper application, which is a bootcamp directory website

## Usage

Rename "config.env.env" to "config.env" and update the values/settings to your own

## Install dependencies

```
npm install
```

## Run app

```
#Run in dev mode
npm run dev

#Run in production mode
npm start
```
